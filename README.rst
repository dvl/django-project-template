=======================
django-project-template
=======================


This project provides an alternative startproject structure with some
boilerplate code for a quick start of your Django Projects.

************
Installation
************

.. code-block:: sh

  $ django-admin.py startproject --extension=txt --name=Makefile,Procfile,.env-example --template=https://github.com/dvl/django-project-template/archive/master.zip my_project
  $ cd my_project
  $ pip install -r requirements.txt
